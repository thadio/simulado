# Simulado Java — POO II

Aplicação em HTML, CSS, JavaScript e PHP com persistência em MySQL.

## Como usar

Configure o arquivo `.env` com os dados do MySQL, publique a pasta em um servidor PHP e acesse `index.php`.

Na primeira execução, a API cria as tabelas necessárias e migra as perguntas para o banco de dados.

## Recursos

- 200 questões
- 10 temas
- 1 questão por vez
- Navegação anterior/próxima
- Seletor de questão 1 a 100
- Perguntas, alternativas, gabarito e justificativas no banco de dados
- Feedback após gravar cada resposta
- Justificativa para alternativas corretas e incorretas
- Timer da tentativa
- Placar com acertos, respondidas e aproveitamento
- Ranking persistido
