# Simulado Java — POO II

Site estático com HTML, CSS e JavaScript, com `index.php` opcional para hospedagem PHP.

## Como usar

Abra `index.html` diretamente no navegador ou publique a pasta em um servidor PHP e acesse `index.php`.

## Recursos

- 100 questões
- 10 temas
- 1 questão por vez
- Navegação anterior/próxima
- Seletor de questão 1 a 100
- Feedback imediato
- Comentário para todas as alternativas
- Pontuação acumulada
